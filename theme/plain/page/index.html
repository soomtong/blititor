Template Engine

Jinja2 스타일의 노드JS 템플릿 엔진 Nunjucks Template Language 를 사용해 자유롭게 HTML 페이지에 내용을 출력하고 파일을 포함할 수 있습니다.

Easy Customize

즉시 사용할 수 있는 다양한 템플릿을 통해 원하는 디자인을 쉽게 만들어 낼 수 있습니다. 지금 보시는 테마는 PureCSS 를 기반으로 만들어져 있습니다.

Module System

가장 강력한 모듈 생태계를 가지고 있는 npm 을 통해 확장 가능합니다. 블리티터가 사용하고 있는 패키지 리스트.
